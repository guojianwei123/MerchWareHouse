# 产品需求：谷子录入与管理

## 目标
用户可以通过图片/截图辅助录入或手动录入谷子，并在确认后维护自己的库存。

## 核心流程
1. 用户上传谷子图片或订单截图。
2. 系统生成可编辑草稿，用户确认分类、IP、人物、系列、价格和物理属性。
3. 用户保存草稿，谷子进入库存。
4. 用户可以查看库存列表，并按 IP、人物、系列、分类筛选。
5. 用户可以编辑或删除已入库谷子。

## 字段要求
- 库存项必须符合 `GuziItem` schema。
- 图片以 `imageUrl` 形式关联到库存项。
- `series` 是基础字段，用于后续统计和筛选。
- 价格字段使用 CNY，物理尺寸使用 mm。

## 成功标准
- 用户可以完成手动录入和 AI 草稿确认入库两条路径。
- 用户可以在列表中筛选并找到指定 IP、人物、系列或分类的谷子。
- 编辑和删除操作只影响目标库存项。
