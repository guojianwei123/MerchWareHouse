# 产品概述：谷子仓库

## 目标用户
购买二次元周边（谷子）的玩家与收藏者，需要管理自己的藏品、空间展示和资金投入。

## MVP 范围
MVP 优先覆盖“录入-管理-统计-展示”的闭环：

1. 建立可校验的谷子分类和价格数据模型。
2. 支持图片/截图辅助录入，并允许用户确认后入库。
3. 支持库存列表、编辑、删除和基础筛选。
4. 基于价格记录生成资产统计看板。
5. 保存展示柜空间布局，并生成只读分享视图。

语音录入、同城交换、市场自动抓价、隐藏款概率统计等能力暂不进入 MVP。

## 核心实体与概念
- **IP**：如“排球少年”、“初音未来”等。
- **人物 (Character)**：IP 下的具体角色。
- **系列 (Series)**：如“十周年系列”、“生日系列”。
- **种类 (Category)**：纸制类、亚克力类、徽章类、布艺类、手办/模型类、数码/实用类、特殊类。
- **谷子 (GuziItem)**：具体的周边单品，包含图片、分类、IP、人物、系列和物理属性。
- **价格记录 (PriceRecord)**：包含官方发售价 (Official Price)、实际购入价 (Purchase Price) 和当前市场价 (Market Price)。
- **展示柜 (Showcase)**：用户保存的线上空间布局，用于展示和分享藏品。

## 核心功能模块
1. **谷子录入与管理**：支持上传谷子图片、AI 草稿确认、手动录入、编辑、删除和列表筛选。
2. **资产统计看板**：展示总投入、当前总估值、官方发售价汇总、各 IP/人物/系列投入占比。
3. **展示柜 (Showcase)**：保存空间布局，生成用户的线上虚拟展示柜供只读分享。
4. **AI 助手**：提供右上角 AI 对话入口，支持识别谷子图片、解释谷子术语，并辅助用户了解 IP、角色和品类相关知识。

## 阶段顺序
1. 数据建模：完成 `GuziItem`、`PriceRecord`、空间节点和展示柜共享数据结构。
2. 库存管理：完成入库、编辑、删除、列表和筛选。
3. AI 摄取：完成图片/截图到可校验草稿数据的 MVP 链路。
4. 空间与配件：完成空间布局、碰撞检测和基础配件推荐。
5. 资产看板：基于库存和价格记录计算资产统计。
6. 展示柜分享：保存布局并提供公开只读展示页。
7. AI 助手导航：新增 AI 对话入口，复用现有 AI 能力完成谷子识别和知识问答。

## 成功标准
- 用户可以新增并管理至少一个带图片、分类、IP、人物、系列和价格信息的谷子。
- 外部输入在进入业务服务前通过 Zod schema 校验。
- 用户可以查看总投入、当前估值以及按 IP/人物/系列聚合的资产占比。
- 用户可以保存展示柜布局，并打开公开只读分享视图。
