# 产品概述：谷子仓库

## 目标用户
购买二次元周边（谷子）的玩家与收藏者，需要管理自己的藏品和资金投入。

## 核心实体与概念
- **IP**：如“排球少年”、“初音未来”等。
- **人物 (Character)**：IP下的具体角色。
- **系列 (Series)**：如“十周年系列”、“生日系列”。
- **种类 (Category)**：吧唧（马口铁徽章）、立牌、吧唧托、毛绒、拍立得等。
- **谷子 (GuziItem)**：具体的周边单品。
- **价格记录 (PriceRecord)**：包含官方发售价 (Official Price)、实际购入价 (Purchase Price) 和当前市场价 (Market Price)。

## 核心功能模块
1. **谷子录入与管理**：支持按上述分类标签上传谷子图片、录入信息。
2. **资产统计看板**：展示总投入（购入价总和）、当前总估值（市场价总和）、各 IP/人物 投入占比。
3. **展示柜 (Showcase)**：生成用户的线上虚拟展示柜供分享。
