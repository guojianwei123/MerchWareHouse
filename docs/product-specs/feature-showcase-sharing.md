# 产品需求：展示柜分享

## 目标
用户可以保存线上展示柜布局，并生成公开只读视图用于分享。

## 核心流程
1. 用户在展示柜编辑器中摆放库存谷子。
2. 系统保存展示柜空间节点和展示项关系。
3. 用户生成分享入口。
4. 访问者打开分享页，只能查看展示柜和谷子基础展示信息。

## 数据要求
- 展示柜包含标题、所有者、空间节点、展示项和公开状态。
- 展示项引用库存 `GuziItem`，但分享视图只暴露名称、IP、人物、系列、分类和图片。
- 分享视图不暴露购入价、市场价等资产信息。

## 成功标准
- 用户可以保存并再次打开展示柜布局。
- 分享页可以在未登录状态下只读访问公开展示柜。
- 私密展示柜不会通过公开链接暴露。
