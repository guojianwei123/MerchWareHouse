# 产品需求：谷子精细化分类与个性化服务

## 业务背景
谷子种类繁多，不同种类的物理属性和收纳方式完全不同。系统需要支持针对不同种类的精细化数据结构，以便后续提供如“收纳配件推荐”、“同城交换”等个性化服务。

## 七大核心分类与特性

### 1. 纸制类 (PaperCard)
- **子类**：海报、明信片、拍立得（小卡）、色纸、贴纸
- **特性**：便宜、易收集、适合“抽卡”
- **核心数据结构需求**：长度 (length)、宽度 (width)
- **衍生服务**：推荐相应尺寸的高透自封袋、卡砖、收纳册。

### 2. 亚克力类 (Acrylic)
- **子类**：立牌、亚克力挂件（钥匙扣）、摆件
- **特性**：展示性强，核心谷子
- **核心数据结构需求**：高度 (height)、是否带底座 (hasBase)
- **衍生服务**：推荐展示防尘盒、防刮膜。

### 3. 徽章类 (Badge)
- **子类**：圆形徽章、闪徽、烫金徽章
- **特性**：收藏交换市场活跃，痛包文化核心
- **核心数据结构需求**：直径 (diameter) 或 形状 (shape)
- **衍生服务**：痛包搭配模拟、吧唧保护套（如 58mm/75mm）自动推荐、同城交换集市。

### 4. 布艺类 (Fabric)
- **子类**：抱枕、挂画、痛包
- **特性**：实用+展示
- **核心数据结构需求**：尺寸、材质说明
- **衍生服务**：清洗保养建议。

### 5. 手办 / 模型类 (Figure)
- **子类**：PVC手办、粘土人、景品
- **特性**：价格高、收藏价值强
- **核心数据结构需求**：比例 (scale, 如 1/7)、高度 (height)、厂商
- **衍生服务**：大型避光展示柜推荐、高单价资产走势分析。

### 6. 数码 / 实用类 (Practical)
- **子类**：手机壳、鼠标垫、卡套
- **特性**：日常可用
- **核心数据结构需求**：适用型号（如 iPhone 15）

### 7. 特殊类 (Special)
- **子类**：盲盒、限定联名周边
- **特性**：稀缺、有盲盒拆箱机制
- **衍生服务**：盲盒端盒记录、隐藏款概率统计。
