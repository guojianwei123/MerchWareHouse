# 产品需求：资产统计看板

## 目标
用户可以基于库存和价格记录了解自己的总投入、当前估值和收藏结构。

## 核心指标
- **总投入**：所有库存项购入价总和。
- **当前总估值**：所有库存项当前市场价总和。
- **官方发售价汇总**：所有库存项官方发售价总和。
- **盈亏估算**：当前总估值减总投入。
- **分组占比**：按 IP、人物、系列、分类聚合购入价和当前估值。

## 数据来源
- `GuziItem` 提供基础分类、IP、人物、系列和当前价格字段。
- `PriceRecord` 保存官方发售价、购入价、当前市场价和记录日期。
- MVP 使用用户手动维护的市场价，不自动抓取外部价格。

## 成功标准
- 用户可以看到总投入、当前总估值、官方发售价汇总和盈亏估算。
- 用户可以查看按 IP、人物、系列、分类聚合的资产占比。
- 空库存时看板展示空状态，不报错。
